# Histoire de la NBA — Site statique (Master 1)

Site statique prêt à déployer sur GitHub Pages. Contenu : HTML5 + CSS3 uniquement.

## Structure

- 6 pages : index, history, teams, players, gallery, contact
- css/style.css

## Déploiement

1. Créer un dépôt GitHub
2. Copier les fichiers et `git add .` / `git commit`
3. Pousser sur `main` puis activer GitHub Pages (branch: main, folder: /)

## Remarques
- Les images sont chargées depuis Unsplash via URL (licence libre/usage non commercial conseillé).
- Aucune logique serveur — le formulaire est fictif (action="#").
- Le projet respecte les contraintes demandées : pseudo-classes, pseudo-éléments, variables CSS, propriétés demandées, Flexbox & Grid.
